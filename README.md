# 📊 Taller 2: Manejo de Listas en Python (Estructuras Lineales)

¡Hola! En este repositorio presento la solución al segundo taller de laboratorio para la asignatura de Algoritmos 1 en Uniremington.

El proyecto consiste en la exploración y aplicación de métodos fundamentales para la manipulación de listas en Python, utilizando una estructura basada en países de Sudamérica.

## 📁 Estructura del Proyecto

El taller está dividido en archivos independientes que resuelven cada punto de la guía de forma limpia:

* **1Creacion_de_lista.py:** Inicialización de la lista indexada con 5 elementos base y consultas directas de posiciones.
* **2Acceder_a_posiciones.py:** Manejo de lecturas por posición y análisis conceptual del error de desbordamiento de rango (`IndexError`).
* **3Agregar_elementos.py:** Uso de métodos para adjuntar elementos al final de la estructura (`.append()`) e insertar en posiciones intermedias específicas (`.insert()`).
* **4Eliminar_elementos.py:** Control de remoción de datos mediante búsquedas por nombre de elemento (`.remove()`) y por extracción de índices (`.pop()`).
* **5Modificar_elementos.py:** Actualización dinámica de valores dentro de la lista mediante asignación directa por índice.
* **6Agregar_multiples_elementos.py:** Carga masiva de datos en lote (`.extend()`) y cálculo del tamaño final de la estructura mediante la función `len()`.
